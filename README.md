# Raytracer

![alt text](https://github.com/oDqnger/raytracer/blob/main/image.png)

Features:  
    Lambert's cosine law  
    Diffuse reflection  
    Specular reflection (still WIP)  
    Shadows  
    Ray-sphere interception  
    Directional and point lights  
    Movable camera
